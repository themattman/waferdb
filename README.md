waferDB.js -- Caching Middleware for JavaScript
==========

A lightweight caching layer for a key-value store.

## usage

`wafer.get('438934')`

## tests

We will have a `test` folder that utilizes tap `https://npmjs.org/package/tap`

## install (eventually)
